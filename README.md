### PR 방법

1. **새로운 Branch 생성**
2. **스터디 repository를 본인 계정에 fork**
3. fork 한 repository를 clone
4. **[계정이름]/README.md를 작성!**
5. 내용을 정리한 README.md 파일을 commit 및 push
6. **웹사이트에서 본인 계정에 fork 된 repository로 이동**
7. **Contribute → Open pull request 로 PR**
